## 🔄 Skill Loading Protocol

1. **Analyze** the task requirements
2. **Select** relevant skills from appropriate categories
3. **Load** skill files from `SKILLS/<category>/`
4. **Apply** guidelines and patterns from skills
5. **Combine** multiple skills if task is complex

**Example:** Building a React app with tests and security:
- Load: `react-best-practices`, `testing-patterns`, `vulnerability-scanner`
